# Welcome

## This Golang-based Slackbot was made to parse and return the newest articles published to Arxiv in an attempt to facilitate easier tracking of Arxiv publishes.

---

# Installation:

1.Clone the repo
2.Export Slack Token
3.run the command:
4.`go run main.go`
5.Add bot to channel
6.Run queries
